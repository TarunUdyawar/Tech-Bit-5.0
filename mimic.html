Typing Titans

Description:

"Mimic Web" is a competitive game where participants showcase their web development skills by cloning given web pages. The game tests their ability to replicate web designs accurately under time constraints. Participants will compete in one round, where they will be given a web page to clone. The objective is to demonstrate advanced web crafting abilities by accurately replicating the design and functionality of the provided web page within the allotted time.

Rounds:

o Objective: Clone a provided web page.
o Duration: 30/45 minutes
o Process:
 Participants will be given a web page to clone.
 Participants must replicate the web page as accurately as possible within the time limit.
o Winner: The participant with the most accurate and precise clone in the shortest time will be declared the winner.

Rules:

• This is an individual competition.
• Participants must use the provided tools and resources to clone the web page.
• The web page clone should match the original as closely as possible in design, layout, and functionality.
• Time limits must be strictly adhered to.
Judging Criteria: • Accuracy of the cloned web page. • Attention to detail in design, layout, and functionality. • Speed in completing the clone within the given time frame. • Overall performance across both rounds.
